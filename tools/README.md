# Automation Domain (tools)

지식 자동 동기화, 파이프라인 가동, 재사용 가능한 빌드 관련 유틸리티 및 스크립트가 수록된 자동화의 핵심 엔진 도메인입니다.

- **`automation/update-docs.py`**: Jina Reader API와 5개 채널 패턴을 통해 외부 문서들을 마크다운으로 파싱, 수집, 증분 업데이트하는 핵심 스크립트.
- **(빌드 스크립트 등 기타 유틸리티)**: 프로젝트 특성에 맞춰 덧붙이거나 커스텀할 수 있는 공통 툴 영역.
- `start.bat`을 통해 일괄 실행 가능.
